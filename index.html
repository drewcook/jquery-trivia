jQuery Trivia Game

About This Project

Hi there. I developed this game solely for the purpose of utilizing the power of jQuery in the browser. jQuery is a very lightweight JavaScript library, with the motto of "write less, do more", and is an efficient way of writing JavaScript more quickly while also making the code a bit easier to read. It can accomplish a lot of common tasks that may take multiple lines of code in written in pure Javascript and wrap all them into methods that can be called with just a single line. It is also great at performing Ajax requests with ease, like pulling in thousands of lines of data from say, a JSON file full trivia questions and their choices and answers!

About The Game

This is a simple trivia game, where you will be presented with multiple choice questions, just like the TV game show 'Who Wants To Be A Millionaire?' Only one of the choices will be the correct answer. There will be a total of 10 questions asked. Each question that is answered correctly will add 100 points to your total score. Try and get 1000 points! At the end of the game, you can view your results as well as choose to play the game again.
